# 🍔 Foodie Express - Android Food Ordering App

Foodie Express is a modern Android-based food ordering application developed using **Kotlin** in **Android Studio**. The app allows users to browse restaurants, explore food menus, add items to cart, and place orders seamlessly with integrated online payment support.

---

## 🚀 Features

* 🔐 User Authentication (Login & Signup)
* 🍽 Restaurant Listing with Images & Ratings
* 🍔 Food Menu with Add to Cart functionality
* 🛒 Cart Management System
* 💳 Razorpay Payment Integration (Test Mode)
* 📦 Order Tracking System
* 🔔 Toast Notifications (Popup Messages)
* 🎨 Modern UI with Material Design
* ⚡ Smooth Navigation & Animations

---

## 📱 App Screens

* Login Screen

## 🛠 Technologies Used

* **Language:** Kotlin
* **IDE:** Android Studio

* **Payment Gateway:** Razorpay (Test Mode)
